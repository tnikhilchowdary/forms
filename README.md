# React Forms Project

This project demonstrates the implementation of a multi-field form using React. It includes various input types such as text, email, radio buttons, file upload, dropdown selection, and checkbox groups. The form uses React state hooks (`useState`) to handle input and submission.

## Features

- Text inputs (First Name, Last Name, Email, Contact)
- Gender selection using radio buttons
- Subject selection using checkboxes
- File upload field for resume
- Dropdown for additional options
- Textarea for about section
- Form validation using `required`
- Submission handler using `onSubmit`
